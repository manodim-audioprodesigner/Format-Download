# 🎵 Format Download

Um aplicativo desktop simples e elegante para baixar áudio e vídeo do YouTube.

![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

## 📋 Funcionalidades

- 🔗 Download de vídeos a partir de uma URL do YouTube.
- 🎵 Conversão para MP3 com qualidade ajustável (128, 256, 320 kbps).
- 🎬 Download de vídeo MP4 em várias qualidades (720p, 1080p, 4K).
- 🖥️ Interface web amigável e responsiva.
- 🔄 Fechamento automático do servidor ao fechar o navegador.
- 🔐 Solicita automaticamente permissão de administrador para funcionamento correto.

## 🚀 Como executar o projeto

### 📋 Pré-requisitos
- [ffmpeg.exe] (https://www.ffmpeg.org/download.html)
- [Python 3.10+](https://www.python.org/downloads/)
- [Git](https://git-scm.com/) (opcional, mas recomendado para clonar)
- **Windows 10 ou 11** (o aplicativo foi desenvolvido e testado para este SO).
